Il est possible de rendre une image semi-transparante en utilisant la propriété "opacity" dans le code CSS du programme. Dépendamment des besoins esthétiques du site Web, la mesure de l'opacité peut varier de 1 (étant 100% d'opacité) et 0 (étant entièrement transparent). Cette propriété peut être appliquée sur les textes, les images, et même le fond de la page.

Dans la plupart des cas, elle est utilisé avec le pseudo-classe ":hover". Dépendamment des propriétés appliquées, ce dernier change minimalement l'apparence d'un élément spécifique de la page lorsque le curseur passe dessus. Pour pouvoir l'utiliser, il faut toujours le précédé de "a" comme ceci: a:hover.

D'autres pseudo-classes pouvant être pratique à appliquer sont ":link", ":visited" et ":active". Tout comme "hover", celles-ci sont utilisées pour modifier le comportement de l'apparence des liens de la pages. Ces pseudo-classes permettent de changer la couleur des liens une fois visité par l'utilisateur et/ou s'ils sont sélectionnés.

En utilisant tous deux ensembles, il est possible de créer un site Web attrayant et interactif pour les utilisateurs.
